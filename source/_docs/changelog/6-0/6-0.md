## 0.0.0
**JulyorAugust Something, 2025**
* Major update to Western Header and main navigation
* Now have option for "Hamburger always on," so main menu is hidden by default
* Major update to footer
    * site footer blocks can generally stay the same, may need to update background colors on image blocks
    * new artwork by local artist Gretchen Leggitt!
* Update to some component styles
    * Regular link styles/animations
    * Call to Action Links
    * Announcements
    * Image with Description

* Theme colors have been updated, the medium green has been removed and yellow has been added back. Light green is still available, but it's use on the web has been reduced.
    * Some colors/gradients have been removed as background options on blocks
* The old set up for "box headlines" has been deprecated, it is now a style variant on banners 
    * This should be easier to use
    * WebTech will handle any conversions from the old set up
    * More style options have also been added for this

* In layout builder, you can now add style classes to SECTIONS. This allows you to have a full width background color/image, and layer other blocks on top
